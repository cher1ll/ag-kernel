# Глоссарий терминов AG-Backtester

## Основные концепции

**AG-Backtester** - Легковесный, детерминистический движок для бэктестирования с C ядром, Rust мостом и Python API.

**Backtesting** - Тестирование торговых стратегий на исторических данных для оценки их потенциальной прибыльности.

**Deterministic** - Детерминистический - всегда дает одинаковые результаты при одинаковых входных данных.

## Walk-Forward Analysis (WFA)

**WFA (Walk-Forward Analysis)** - Метод тестирования стратегий с использованием скользящих окон оптимизации и валидации.

**IS (In-Sample)** - Период обучения/оптимизации, на котором подбираются лучшие параметры стратегии.

**OOS (Out-of-Sample)** - Период валидации, на котором тестируется стратегия с уже найденными параметрами.

**Window** - Окно данных, состоящее из IS и OOS периодов.

**Parameter Grid** - Сетка параметров для оптимизации (все возможные комбинации значений параметров).

## Метрики производительности

**PF (Profit Factor)** - Коэффициент прибыльности, отношение общей прибыли к общим убыткам.

**Return** - Доходность в процентах от начального капитала.

**Sharpe Ratio** - Коэффициент Шарпа, риск-скорректированная доходность.

**Max DD (Maximum Drawdown)** - Максимальная просадка капитала в процентах.

**Win Rate** - Процент прибыльных сделок.

**Consistency** - Процент прибыльных OOS окон в WFA.

**Robustness Score** - Оценка устойчивости стратегии.

**MVS (Minimum Viable Strategy)** - Минимально жизнеспособная стратегия (проверка базовых критериев).

## Технические индикаторы

**Bollinger Bands** - Полосы Боллинджера (скользящая средняя ± стандартные отклонения).

**RSI (Relative Strength Index)** - Индекс относительной силы (осциллятор от 0 до 100).

**Z-Score** - Стандартизированное отклонение цены от среднего значения.

**EMA (Exponential Moving Average)** - Экспоненциальная скользящая средняя.

**ATR (Average True Range)** - Средний истинный диапазон (мера волатильности).

**ADX (Average Directional Index)** - Индекс направленного движения (сила тренда).

## Торговые концепции

**Tick** - Минимальное изменение цены инструмента.

**Tick Size** - Размер тика (минимальный шаг цены).

**Spread** - Разница между ценой покупки и продажи.

**Slippage** - Проскальзывание цены при исполнении ордера.

**Position Size** - Размер позиции (доля капитала на сделку).

**Risk Percent** - Процент риска от цены входа.

**Reward Ratio** - Соотношение потенциальной прибыли к риску.

**Stop Loss (SL)** - Стоп-лосс, ордер на закрытие убыточной позиции.

**Take Profit (TP)** - Тейк-профит, ордер на фиксацию прибыли.

## Типы ордеров и сделок

**Market Order** - Рыночный ордер (исполняется немедленно по текущей цене).

**Limit Order** - Лимитный ордер (исполняется по указанной цене или лучше).

**LONG** - Длинная позиция (покупка с расчетом на рост цены).

**SHORT** - Короткая позиция (продажа с расчетом на падение цены).

**BUY/SELL** - Покупка/Продажа.

## Данные и форматы

**aggTrades** - Агрегированные сделки Binance (timestamp, price, qty, is_buyer_maker).

**OHLC** - Open, High, Low, Close (данные в формате свечей).

**Bucket** - Временной интервал для агрегации данных.

**Parquet** - Бинарный формат хранения данных с ZSTD сжатием.

**CSV** - Текстовый формат данных (Comma-Separated Values).

## Объемные индикаторы

**Volume Delta** - Разница между объемами покупок и продаж.

**Aggressive Volume** - Объем агрессивных (рыночных) ордеров.

**Buy/Sell Volume** - Объем покупок/продаж.

**Volume Spike** - Всплеск объема торгов.

## Стратегические концепции

**Mean Reversion** - Возврат к среднему (стратегии, основанные на отклонении от среднего).

**Momentum** - Импульс (стратегии, следующие за трендом).

**Breakout** - Пробой уровня поддержки/сопротивления.

**Pullback** - Откат цены в рамках тренда.

**Squeeze** - Сжатие волатильности (низкая волатильность перед движением).

**Regime Switch** - Смена режима рынка (тренд/флет).

## Архитектурные термины

**C Kernel** - Ядро на языке C (слой исполнения).

**Rust Bridge** - Мост на Rust (безопасные FFI привязки).

**Python API** - Пользовательский интерфейс на Python.

**FFI (Foreign Function Interface)** - Интерфейс для вызова функций из других языков.

**PyO3** - Библиотека для создания Python расширений на Rust.

**Maturin** - Инструмент для сборки Python пакетов с Rust кодом.

## Оптимизация производительности

**Batch Processing** - Пакетная обработка (обработка массивов данных за один вызов).

**Zero-Copy** - Операции без копирования данных.

**Vectorized Operations** - Векторизованные операции (pandas/numpy).

**RAII (Resource Acquisition Is Initialization)** - Автоматическое управление ресурсами в Rust.

## Временные фильтры

**Asian Session** - Азиатская торговая сессия (низкая ликвидность).

**Time Filter** - Временной фильтр (избегание определенных часов торговли).

**Bucket MS** - Размер временного интервала в миллисекундах.

## Адаптивные методы

**ATR Adaptive** - Адаптивные стопы на основе ATR.

**Volatility Sizing** - Размер позиции на основе волатильности.

**Dynamic SL/TP** - Динамические стоп-лосс и тейк-профит.

## Статистические термины

**Monte Carlo** - Метод статистического моделирования.

**Statistical Significance** - Статистическая значимость результатов.

**Overfitting** - Переобучение (слишком точная подгонка под исторические данные).

**Robustness** - Устойчивость стратегии к изменениям параметров.

## Файловая структура

**Userland** - Пользовательские утилиты и вспомогательные функции.

**Tearsheet** - Отчет о производительности стратегии с графиками.

**Equity Curve** - Кривая капитала (изменение капитала во времени).

**Hypothesis (H01, H02, ...)** - Гипотеза стратегии (нумерованные стратегии для тестирования).
